# mrm-task-tsx

[Mrm](https://github.com/sapegin/mrm) task that adds [tsx](https://github.com/esbuild-kit/tsx).

## What it does

- Adds `"start" : "tsx ./src/index.ts"` to `package.json` scripts
- Installs the devDependency `tsx`

## Usage

```
npx mrm tsx
```
